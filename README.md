 A lightweight browser extension that allows you edit any text on a webpage, much like using inspect element, but much simpler. 

**INSTALLATION AND USAGE GUIDE**
1) Download the file above depending on what browser you use (Chromium based or Firefox based). Alternatively, download from https://mazebeamtech.github.io/MazeEdit/
2) For Chromium-based browsers, turn on developer mode in the extensions page and upload zip file. For Firefox-based browsers, head to about:config and search xpinstall.signatures.required. Toggle it to **false**.
3) Pin the extension to the toolbar, and click to enable. Your webpage now turns into an editable document. Enjoy!
